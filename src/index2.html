BTC Relay is a building block that allows Ethereum contracts to securely verify Bitcoin transactions without any centralized parties.

Features

Open Source Code

Anyone can verify the source: Github Repo. Questions? Ask in the Gitter

Decentralized

Community members, Relayers, are incentivized to provide the service. Anyone can be a Relayer: no special hardware or electricity costs like Bitcoin mining.

Easy to Use

In just a few lines of code in a DApp or smart contract, BTC Relay is easy to integrate.
